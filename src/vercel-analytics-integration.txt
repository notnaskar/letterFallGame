1. Install the analytics package: npm install @vercel/analytics
2. In your main entry file (main.jsx), add:

import { Analytics } from '@vercel/analytics/react';

Then, render <Analytics /> at the root level, e.g.:

import React from 'react';
import ReactDOM from 'react-dom/client';
import App from './App';
import { Analytics } from '@vercel/analytics/react';

ReactDOM.createRoot(document.getElementById('root')).render(
  <React.StrictMode>
    <Analytics />
    <App />
  </React.StrictMode>
);

3. Deploy your app to Vercel and enable Analytics in the Vercel dashboard.

Reference: https://vercel.com/docs/analytics/quickstart https://stackoverflow.com/questions/77790865/how-to-enable-vercel-web-analytics-in-react-app
